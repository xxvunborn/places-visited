## places visited

Fullstack work for traninig back end frontend.

# Backend

- create the `.env` from `env.sample`
- run mongodb or add a enviroment from mongodb.com

for run the backend:

`npm start`

# Frondend

- create the `.env` from `env.sample`
- add enviroment variable for mapbox (mapbox.com)

Run the frontend:

`yarn start`
